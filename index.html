Introduction

Spotifyd aims to be a suckless alternative to the official spotify client for linux. It uses the same client/server model as the popular MPD. This means that you can have a server running on one computer, a raspberry pi for example, and the control it from any other machines on the network. Neat huh? It's very lightweight and will run on basically any hardware you can imagine.

Installation

This couldn't be easier. A simple:

make && sudo make install

Will install spotifyd.

Configuration

A simple configuration file is read from $HOME/.spotifyd.rc. It looks like this:

username my_user_name
password my_password
unix-socket /tmp/spotifyd
port 12345

The server can be configured to listen on a unix socket, a network socket, or both.

Dependencies

Spotifyd depends on libspotify and alsa to be installed (this basically means that you must be running linux).
